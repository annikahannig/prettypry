# PrettyPry

Adds pry configuration (like jazz_hands gem) but
working with ruby >= 2.0

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'prettypry', :github => 'trustner/prettypry'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install prettypry
